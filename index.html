OpenAI‑Compatible, Multi‑Provider Gateways (handy for DeepSeek & Gemini)

  • LiteLLM
    One API format → 100+ providers (proxy/server + SDK). Great for swapping models without rewriting code.

    Run a tiny gateway that forwards /chat/completions to many backends.

Local / In‑Browser (no server)

  • WebLLM (GitHub)
    Run models in the browser with WebGPU; OpenAI‑compatible API shim for simple integrations.
  • WebLLM – Home / Docs
    Docs and quickstart; privacy‑friendly and bandwidth‑light.

Tips for Low‑End Browsers

  • Prefer JSON endpoints; avoid loading big JS bundles/frameworks.
  • Stream text when possible (SSE) but fall back to plain JSON if SSE isn’t supported.
  • If CORS blocks you, call through a tiny proxy (e.g., LiteLLM Proxy or a Cloudflare Worker).
